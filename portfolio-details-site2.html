Site Profissional para Loja de Bicicletas

Desenvolvimento de um site completo e profissional para uma loja de bicicletas, incluindo catálogo de produtos e sistema de contato. O projeto incluiu:

  • Design moderno e atrativo
  • Catálogo de produtos organizado
  • Sistema de busca e filtros
  • Páginas de detalhes dos produtos
  • Formulário de contato e orçamento
  • Seção de serviços e manutenção
  • Blog com dicas para ciclistas
  • Integração com redes sociais

Tecnologias Utilizadas

  • HTML5 semântico
  • CSS3 com animações
  • TypeScript para funcionalidades
  • Tailwind CSS para layout responsivo
  • PHP para backend simples
  • MySQL para banco de dados
